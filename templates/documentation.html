Documentation

Rook Hub is a project built by Aaron Omale, from Nigeria. As a final project for Harvard's Introduction to Computer Science course on EdX platform. I came about the idea for this site because as a rookie developer myself, I rarely see quiz sites to test my knowledge on what I've been studying. The questions on this site will challenge rookies to study harder, and also boost their confidence.


Add your own questions!

Rook hub provides you the user with the users with an opportunity to add specific questions to the database through an in-built content management system (CMS). But for security reasons the route is not available to the public. To add questions, Send Email to request for access or email your questions directly. {% endblock %}
